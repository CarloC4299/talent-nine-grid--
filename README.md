# 🎯 人才九宫格 | Talent Nine-Grid

基于"业绩-潜力"二维模型的人才盘点可视化工具，快速识别人才分布、制定差异化培养策略。

## 功能特点

- **九宫格可视化**：将员工按业绩和潜力两个维度映射到九宫格中，直观展示人才分布
- **交互操作**：支持拖拽、缩放，灵活调整人才位置
- **分类统计**：自动统计各象限人数及占比
- **导出功能**：盘点结果可导出为Excel，便于后续分析

## 文件说明

| 文件 | 说明 |
|------|------|
| `人才九宫格可视化.html` | 九宫格可视化主程序（浏览器直接打开） |
| `人才盘点工具.xlsx` | 人才盘点数据模板（用于录入和整理盘点数据） |

## 使用方式

1. 在 `人才盘点工具.xlsx` 中填入员工盘点数据
2. 用浏览器打开 `人才九宫格可视化.html`
3. 导入数据或手动添加人才信息
4. 查看九宫格分布，制定培养和发展计划

## 技术栈

- 纯前端实现（HTML + CSS + JavaScript）
- 无需服务器，浏览器直接运行
